## Overview

Fly Web is the browser-based interface for managing your JFrog Fly workspace. It provides a unified view of all your artifacts, releases, workflows, and team settings.

<Info>
  Access Fly Web at [fly.jfrog.ai](https://fly.jfrog.ai) or your custom subdomain.
</Info>

## Quick Actions

### From the Dashboard

<Steps>
  <Step title="Connect Repository">
    Click **+ Connect Repository** to add a new Git repository
  </Step>
  
  <Step title="Create Release">
    Click **Create Release** to manually create a new release
  </Step>
  
  <Step title="Invite Team Member">
    Click **Invite** in the team section to add collaborators
  </Step>
</Steps>

### Command Palette

Press `Cmd/Ctrl + K` to open the command palette for quick navigation:

```
Search artifacts...
Go to Releases
Open Settings
Connect Repository
Create Access Token
```

## Search

### Global Search

The search bar supports:

- **Artifact names**: `@myteam/utils`
- **Version numbers**: `2.1.0`
- **Authors**: `@developer`
- **Commit hashes**: `abc123`
- **Natural language**: `authentication library from last week`

### Search Filters

Refine searches with filters:

```
type:npm @myteam/utils
type:docker created:>2024-01-01
author:john@example.com
tag:production
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `Cmd/Ctrl + /` | Focus search |
| `G then D` | Go to Dashboard |
| `G then A` | Go to Artifacts |
| `G then R` | Go to Releases |
| `G then S` | Go to Settings |
| `?` | Show keyboard shortcuts |

## Mobile Access

Fly Web is responsive and works on mobile devices:

- View artifacts and releases
- Check workflow status
- Receive and manage notifications
- Basic team management

<Note>
  For the best experience with complex operations like repository configuration, use a desktop browser.
</Note>
